# Sistema de Parcelamento com Juros

Este projeto em Python calcula o valor total de um pedido com base na quantidade de parcelas e na taxa de juros aplicada.

## Funcionalidades
- Entrada do valor do pedido  
- Escolha da quantidade de parcelas  
- Cálculo automático do valor final com juros  
- Exibição do valor mensal e do valor total

## Tecnologias
- Python 3  
- Execução via terminal

## Como executar
```bash
python parcelamento_juros.py
